While Stephan is running with cargo, Nicola and Sophia invnted the game with boxes.

At the beginning of the game several black and white balls are placed in the box. Then the robots for each move take the ball out of the box and put the ball back of the opposite color. The winner is the one who pulls the white ball on the Nth step.

The robots don't like indeterminacy and want to know the probability of a white ball on Nth step. The probability is a value between 0 (0% chance or will not happen) and 1 (100% chance or will happen). The result is a float from 0 to 1 with two digits precision.

You are given a start set of balls as a string that contains "b" (black) and "w" (white) and the number of the step (N). The order of the balls does not matter.

Input: The start sequence of the balls (string) and the step's number (int).

Output: The probability for white ball, a float.

Example: 

checkio('bbw', 3) == 0.48
checkio('wwb', 3) == 0.52
checkio('www', 3) == 0.56
checkio('bbbb', 1) == 0
checkio('wwbb', 4) == 0.5
checkio('bwbwbwb', 5) == 0.48
